# Python To-Do List

A simple command-line based To-Do List application developed using Python. This project allows users to add, view, and manage daily tasks while demonstrating fundamental Python concepts such as file handling, loops, conditional statements, and list operations.

## Features

- Add new tasks
- View saved tasks
- Remove tasks
- Persistent task storage using text files
- Menu-driven command line interface

## Technologies Used

- Python 3

## Concepts Covered

- File Handling
- Loops
- Conditional Statements
- Lists
- String Handling
- Functions (optional if added later)
